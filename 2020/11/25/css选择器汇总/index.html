CSS选择器汇总

CSS选择器功能非常全面，但常用的也就几个，特别是现在MVVM框架的盛行，不需要经常操作DOM也就用得更少了。

选择器 示例 示例说明 CSS
.class (opens new window) .intro 选择所有class="intro"的元素 1
#id (opens new window) #firstname 选择所有id="firstname"的元素 1
* (opens new window) * 选择所有元素 2
element (opens new window) p 选择所有p元素 1
element,element (opens new window) div,p 选择所有div元素和p元素 1
element element (opens new window) div p 选择div元素内的所有p元素 1
element>element (opens new window) div>p 选择所有父级是 div 元素的 p 元素 2
element+element (opens new window) div+p 选择所有紧跟在 div 元素之后的第一个 p 元素 2
[attribute] (opens new window) [target] 选择所有带有target属性元素 2
[attribute=value] (opens new window) [target=-blank] 选择所有使用target="-blank"的元素 2
[attribute~=value] (opens new window) [title~=flower] 选择标题属性包含单词"flower"的所有元素 2
[attribute|=language] (opens new window) [lang|=en] 选择 lang 属性等于 en，或者以 en- 为开头的所有元素 2
:link (opens new window) a:link 选择所有未访问链接 1
:visited (opens new window) a:visited 选择所有访问过的链接 1
:active (opens new window) a:active 选择活动链接 1
:hover (opens new window) a:hover 选择鼠标在链接上面时 1
:focus (opens new window) input:focus 选择具有焦点的输入元素 2
:first-letter (opens new window) p:first-letter 选择每一个p元素的第一个字母 1
:first-line (opens new window) p:first-line 选择每一个p元素的第一行 1
:first-child (opens new window) p:first-child 指定只有当p元素是其父级的第一个子级的样式。 2
:before (opens new window) p:before 在每个p元素之前插入内容 2
:after (opens new window) p:after 在每个p元素之后插入内容 2
:lang(language) (opens new window) p:lang(it) 选择一个lang属性的起始值="it"的所有p元素 2
element1~element2 (opens new window) p~ul 选择p元素之后的每一个ul元素 3
[attribute^=value] (opens new window) a[src^="https"] 选择每一个src属性的值以"https"开头的元素 3
[attribute$=value] (opens new window) a[src$=".pdf"] 选择每一个src属性的值以".pdf"结尾的元素 3
[attribute*=value] (opens new window) a[src*="runoob"] 选择每一个src属性的值包含子字符串"runoob"的元素 3
:first-of-type (opens new window) p:first-of-type 选择每个p元素是其父级的第一个p元素 3
:last-of-type (opens new window) p:last-of-type 选择每个p元素是其父级的最后一个p元素 3
:only-of-type (opens new window) p:only-of-type 选择每个p元素是其父级的唯一p元素 3
:only-child (opens new window) p:only-child 选择每个p元素是其父级的唯一子元素 3
:nth-child(n) (opens new window) p:nth-child(2) 选择每个p元素是其父级的第二个子元素 3
:nth-last-child(n) (opens new window) p:nth-last-child(2) 选择每个p元素的是其父级的第二个子元素，从最后一个子项计数 3
:nth-of-type(n) (opens new window) p:nth-of-type(2) 选择每个p元素是其父级的第二个p元素 3
:nth-last-of-type(n) (opens new window) p:nth-last-of-type(2) 选择每个p元素的是其父级的第二个p元素，从最后一个子项计数 3
:last-child (opens new window) p:last-child 选择每个p元素是其父级的最后一个子级。 3
:root (opens new window) :root 选择文档的根元素 3
:empty (opens new window) p:empty 选择每个没有任何子级的p元素（包括文本节点） 3
:target (opens new window) #news:target 选择当前活动的#news元素（包含该锚名称的点击的URL） 3
:enabled (opens new window) input:enabled 选择每一个已启用的输入元素 3
:disabled (opens new window) input:disabled 选择每一个禁用的输入元素 3
:checked (opens new window) input:checked 选择每个选中的输入元素 3
:not(selector) (opens new window) :not(p) 选择每个并非p元素的元素 3
::selection (opens new window) ::selection 匹配元素中被用户选中或处于高亮状态的部分 3
:out-of-range (opens new window) :out-of-range 匹配值在指定区间之外的input元素 3
:in-range (opens new window) :in-range 匹配值在指定区间之内的input元素 3
:read-write (opens new window) :read-write 用于匹配可读及可写的元素 3
:read-only (opens new window) :read-only 用于匹配设置 "readonly"（只读） 属性的元素 3
:optional (opens new window) :optional 用于匹配可选的输入元素 3
:required (opens new window) :required 用于匹配设置了 "required" 属性的元素 3
:valid (opens new window) :valid 用于匹配输入值为合法的元素 3
:invalid (opens new window) :invalid 用于匹配输入值为非法的元素 3
