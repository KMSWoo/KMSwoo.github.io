# 经过

每天最令人愉快的事情莫过于deploy之后看着代码顺利的跑起来。但今天虽然是顺利跑起来了，但是dev环境下网站的样式跟部署后的样式有一点点的不一样。

熟练的打开Chrome DevTool最终发现线上版本比dev版本多一条样式，npm run dev 跟npm run build出来的版本不可能是不一样的，所以我第一反应就是应该是浏览器缓存的问题。

遂清除了浏览器缓存之后，发现还是bug依旧在。换浏览器也是一样。

当所有可能的都排除了之后，剩下最不可能的就是答案了。也就是npm run dev 跟npm run build的确是不一样的，最后仔细查看了代码，发现问题在于主组件的样式没有写scoped影响了子组件，问题总算是找到了。

但问题是同一份源码，为什么npm run dev 跟npm run build出来的代码不一样，百思不得其解，约莫是dev的环境默认给你设置了什么参数吧，不得而知。这恐怕得等有时间去仔细深入研究一下源代码才会知道了吧。
