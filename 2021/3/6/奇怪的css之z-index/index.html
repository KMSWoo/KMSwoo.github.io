# 属性定义及使用说明

z-index 属性指定一个元素的堆叠顺序。

拥有更高堆叠顺序的元素总是会处于堆叠顺序较低的元素的前面。

注意： z-index 进行定位元素(position:absolute, position:relative, or position:fixed)。

# 基本规律

因为z-index 仅适用于定位元素，没有定位就没有默认值。定位时的默认值为z-index: auto，可以设置正整数，也可以设置为负整数，也可以是0。

如果有多个定位元素z-index: auto或z-index: 0他们将按照 ：所有“z-索引：AUTO”或“z-索引：0”的后缀，按树顺序排列。这意味着文档中定义的顺序，最后一个元素是最高层。

如果是定位的元素与不定位的元素：定位的元素不设置（即为auto）z-index 在不定位的元素上面（0也在上面），只要设置了负数就在下面

# z-index: 0与z-index: auto的区别

设置z-index: 0或z-index: auto，文档流中后出现的会覆盖先出现的，也就是说z-index: 0与z-index不设置，也就是z-index: auto在同一层级内没有高低之分。

但z-index为0是会创建新的层叠上下文，新的上下文中哪怕再高的z-index值也只能影响当前上下文。所以没有创建层叠上下文的元素同其父级处于一个层叠上下文，也就是说当前的z-index:5有可能在z-index:999上面。

# 总结

  • 不设置z-index值时，默认是auto。默认层也就是0层。
  • z-index: 0 与没有定义 z-index 也就是z-index: auto在同一层级内没有高低之分，文档流中后出现的会覆盖先出现的。
  • z-index: 0 会创建层叠上下文z-index: auto不会创建层叠上下文。
